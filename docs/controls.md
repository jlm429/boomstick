# Controls

| Input           | Action                                |
| --------------- | ------------------------------------- |
| Click arena     | Capture pointer and enable mouse look |
| `W` `A` `S` `D` | Move                                  |
| Mouse           | Look                                  |
| `Space`         | Jump                                  |
| `Esc`           | Pause and release pointer lock        |

The main menu and pause dialog work without pointer lock. Mouse and keyboard are recommended for this prototype.
